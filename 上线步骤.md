# Lester Shi 网站上线步骤(纯网页操作版)

全程免费,不需要命令行。仓库已建好的话从第二步开始。

## 一、建仓库(已完成可跳过)

github.com → 右上角 + → New repository → 名字: `MizuNoNakaniwa.github.io` → Public → Create repository

## 二、上传文件

1. 把 zip 解压到电脑上一个文件夹
2. 打开你的仓库页面 → 点 **Add file** → **Upload files**
3. 把解压出来的**全部内容**(包括 .github 文件夹)拖进上传区
   - 注意: 要拖文件夹里的内容,不是拖 zip 本身
   - 如果 .github 这种点开头的文件夹拖不上去(有些系统隐藏它),单独进去把里面的 hugo.yml 按路径 `.github/workflows/hugo.yml` 传(上传界面能建文件夹路径)
4. 底下 Commit changes 直接点绿色按钮

主题不用你管——部署脚本会自动下载 MemE v5.0.0。

## 三、开启 Pages

仓库页面 → **Settings** → 左栏 **Pages** → Build and deployment → Source 选 **GitHub Actions**

## 四、触发构建

回到仓库 → **Actions** 标签页 → 如果没有自动开始,点左边 Deploy Hugo site to Pages → Run workflow

绿勾出现后(约 1-2 分钟),访问: **https://mizunonakaniwa.github.io**

## 日常发文章

仓库 → content/posts/ → Add file → Create new file → 文件名如 `my-post.md`,开头写:

```yaml
---
title: "文章标题"
date: 2026-07-06
categories: ["分类名"]
tags: ["标签"]
---
```

正文用 markdown,提交即自动发布。转载 X/TikTok 的嵌入写法见 hello-world.md 示例。

## 改社交链接

data/Socials.toml 里把 YOUR_HANDLE 换成你的账号,不要的平台删掉整段。
